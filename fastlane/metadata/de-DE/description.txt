Evaluate Day ist eine elegante und flexible Anwendung für die Verfolgung von allen Parametern Ihres Lebens, Aufzeichnung von Tagesbücher und Analysen aller Lifestyle-Indikatoren.

ALLES UNTER KONTROLLE
Erstellen Sie Ihre einzigartigen Karteikarten, sortieren sie diese und notieren Sie jeden Tag alle Änderungen. Sie erhalten ein objektives Bild über Ihre Fortschritte; Sie werden alle Änderungen in Ihrem Tagesablauf mitbekommen und können rechtzeitige Korrekturen vornehmen.

KARTEIKARTEN
- Farbe - schafft eine Stimmung und charakterisiert verschiedene Aspekte Ihres Lebens
- Ihren Standort -  speichern Sie die Orte, die Sie besuchen
- Phrase - einen kurzen und prägnanten Spruch oder Idee, die nicht mehr so schnell verloren geht
- 100-Punkten-Bewertungskriterien - bewerten Sie allen Eckdaten aus Ihrem Leben auf einer 100-Punkte-Skala
- 10-Punkten-Bewertungskriterien - oder auf einer 10-Punkte-Skala
- 3-Punkten-Bewertungskriterien - Sie können das auch anhand einer 3-Punkte-Skala machen; Es gibt eine bequeme Skala für jeden Zweck.
- Zähler - er kann die Anzahl der verschiedenen Parameter aufnehmen. Sie können Ihren täglichen Zähler einstellen(z. B. getrunkenen Wasser pro Tag) oder summieren Sie Daten über die Zeit  (z. B. Ersparnisse)
- Tracker - verfolgen und markieren Sie verschiedene periodische Aktivitäten
- Gewohnheit - formieren Sie Ihre Gewohnheiten mit Hilfe der kontinuierlichen Überwachung und Erinnerungen
- Liste - halten Sie die Listen fest; zum Beispiel Einkäufen oder Aktivitäten
- Ziel - Sie können auch ein Ziel setzen und ihre Fortschritte verfolgen.
- Journal - führen Sie ein Tagebuch, notieren Sie Ideen oder schreiben Sie ein Buch,
